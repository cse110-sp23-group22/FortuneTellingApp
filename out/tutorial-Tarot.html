TarotDisplay Module

author: Chris Kim date: 06/07/2023

The TarotDisplay is an important part of the Tarot Card function as it
fetches the card data and appends the corresponding card image which
displays upon flip. Then, the description is displayed below the card.

  • Once the date is selected, the output is mapped from the TarotCardAnswer.JSON file.
